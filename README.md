# agent-skills

A collection of agent skills I find useful, organized by category.

## Investing

Skills for researching and screening publicly traded stocks.

- **[fundamental-analysis](Investing/Skills/fundamental-analysis)** — Analyzes a single company's fundamentals and produces a 0–100 "worth owning long term" score, with a beginner-friendly report explaining every metric along the way.
- **[fundamental-screener](Investing/Skills/fundamental-screener)** — Batch-scores a list of tickers (e.g. a watchlist) on the same 0–100 scale and returns a single ranked comparison table instead of full reports.
